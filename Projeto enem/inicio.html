Sobre o site

Projeto Enem é um web site para todos aqueles que desejam um melhor desempenho na hora de realizar o tão esperado e temido Exame Nacional do Ensino Médio. E não é para menos, não é? Afinal, o ENEM é uma das provas mais importantes quando o assunto é nosso futuro. O Projeto Enem visa ajudar-lhe a se preparar da melhor maneira possível para realização da prova.
